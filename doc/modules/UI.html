UI.Dial.new (x, y, size, value, min_value, max_value, rounding, start_value, markers, units, title)
Create a new Dial object.

Parameters:

  • x X position, defaults to 0.
  • y Y position, defaults to 0.
  • size Diameter of dial, defaults to 22.
  • value Current value, defaults to 0.
  • min_value Minimum value, defaults to 0.
  • max_value Maximum value, defaults to 1.
  • rounding Sets precision to round value to, defaults to 0.01.
  • start_value Sets where fill line is drawn from, defaults to 0.
  • markers Array of marker positions.
  • units String to display after value text.
  • title String to be displayed instead of value text.

Returns:

    Instance of Dial.
