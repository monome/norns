new (x_min, x_max, x_warp, y_min, y_max, y_warp, style, show_x_axis, show_y_axis)
Create a new Graph object. All arguments optional.

Parameters:

  • x_min Minimum value for x axis, defaults to 0.
  • x_max Maximum value for x axis, defaults to 1.
  • x_warp String defines warping for x axis, accepts "lin" or "exp", defaults to "lin".
  • y_min Minimum value for y axis, defaults to 0.
  • y_max Maximum value for y axis, defaults to 1.
  • y_warp String defines warping for y axis, accepts "lin" or "exp", defaults to "lin".
  • style String defines visual style, accepts "line", "point", "spline", "line_and_point", "spline_and_point" or "bar", defaults to "line".
  • show_x_axis Display the x axis if set to true, defaults to false.
  • show_y_axis Display the y axis if set to true, defaults to false.

Returns:

    Instance of Graph.

add_point (px, py[, curve[, highlight[, index]]])
Add a point to the graph.

Parameters:

  • px Point's x position.
  • py Point's y position.
  • curve Curve of previous line segment, accepts "lin", "exp" or a number where 0 is linear and positive and negative numbers curve the envelope up and down, defaults to "lin". (optional)
  • highlight Highlights the point if set to true, defaults to false. (optional)
  • index Index to add point at, defaults to the end of the list. (optional)

add_function (func[, sample_quality[, index]])
Add a new y function to the graph. Add a function that will be used to generate a line.

Parameters:

  • func A function that should return the value of y for any given value of x. For example, 'function(x) return math.sin(x) end'
  • sample_quality Quality to sample the graph function at. Values less than 1 will sample less than once per pixel, values higher than 1 can help reduce jitter of graphs with high frequency changes at the cost of performance. Recommended range 0.25 to 4, defaults to 1. (optional)
  • index Index to add function at, defaults to the end of the list. (optional)

  • sample_quality Quality to sample the graph function at, defaults to 1. (optional)
